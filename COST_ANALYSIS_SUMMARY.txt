   Impact: $18,000 saved over 5 years

   ├─ App Platform included

   GCP Cloud Run: $10-20/month
   ├─ Slightly more expensive
   ├─ Better auto-scaling
   ├─ More reliable for production
   └─ Good if using GCP database anyway

   AWS EC2: $20-50/month
   ├─ More expensive for small apps
   ├─ Better for complex workloads
   ├─ Steeper learning curve
   └─ Overkill for 5 agents

   RECOMMENDATION: DigitalOcean (simplicity + cost)
