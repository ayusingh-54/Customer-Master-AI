DATA FLOW DIAGRAM
════════════════════════════════════════════════════════════════════════════

Agent Request Flow:

1. OWC User asks in Claude Code:
   "Find duplicate customers in our database"

2. Claude calls your agent:
   agent = DeduplicationAgent()
   result = agent.find_duplicates()

3. Agent validates license (calls YOUR API):
   POST https://api.yourcompany.com/validate-license
   {"api_key": "lic-owc-xxx", "agent": "deduplication"}

   YOU respond:
   {"valid": true, "expires": "2025-03-13"}

4. Agent queries OWC's Oracle:
   SELECT * FROM hz_parties WHERE status = 'A'

5. Agent uses OWC's Claude:
   POST https://api.openai.com/v1/messages
   (using OWC's Claude API key)

6. Agent returns results to OWC:
   Duplicate groups found, similarity scores, recommendations

7. Agent logs usage (calls YOUR API):
   POST https://api.yourcompany.com/track-usage
   {"api_calls": 1, "tokens_used": 2500, "status": "success"}

   YOU log this for billing

8. You invoice OWC monthly:
   "Used agents 127 times this month. License: $1,250"

THE GOLDEN METRIC
════════════════════════════════════════════════════════════════════════════

NRR (Net Retention Rate) = (Beginning MRR + Expansion - Churn) / Beginning MRR

Target: > 100% (customers pay more over time)

Example:
├─ Month 1 MRR: $1,250 (OWC)
├─ Month 2: OWC asks for 2 more agents (expansion = +$500)
├─ Month 2: No churn
├─ Month 2 MRR: $1,750
├─ NRR = (1,250 + 500) / 1,250 = 140%
└─ You're growing faster than customer acquisition! 🚀
